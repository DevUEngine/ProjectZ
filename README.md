# Neon Requiem

**Cyberpunk Gothic Bullet Heaven Roguelike** built in Godot 4.7.

> Fight through the neon-soaked ruins of Neo-Vesper. Upgrade your cyber-implants. Survive the Eldritch Protocol.

## Platforms
- Windows
- Linux
- macOS
- Web (HTML5)
- Android

## Controls
| Action | Key | Gamepad |
|--------|-----|---------|
| Move | WASD | Left Stick |
| Dodge | Space | X (PS) / A (Xbox) |
| Pause | Escape | Start |

## Core Loop
1. Survive 20-minute runs
2. Collect XP orbs → Level up → Pick weapons & passives
3. Kill bosses every 5 minutes
4. Collect Souls → Permanent upgrades between runs
5. Unlock new characters & weapons

## Weapons (8 base + evolutions)
- Neon Blade (melee, pierce)
- Plasma Pistol (ranged, auto-target)
- Tesla Coil (aura, chain)
- Cyber-Drones (orbitals)
- Eldritch Beam (laser, penetration)
- Scattergun (shotgun, spread)
- Void Grenade (AoE, delayed)
- Nano Swarm (homing, DoT)

## Enemies
- Cyber-Zombie (melee horde)
- Corp Cultist (ranged)
- Aerial Drone (fast, dashes)
- Eldritch Spawn (tank, slow)
- Boss: The Administrator (Phase 1-3)

## Build
Requires Godot 4.7.1+.

```bash
godot --export-release "Windows Desktop" build/neon_requiem.exe
godot --export-release "Web" build/web/index.html
```
